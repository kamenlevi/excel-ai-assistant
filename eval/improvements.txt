To improve the handling of formulas, use the `getUsedRange` method to determine the last row with data.
```javascript
const sheet = workbook.worksheets.getActiveWorksheet();
const usedRange = sheet.getUsedRange();
usedRange.load("rowCount");
await context.sync();
const lastRow = usedRange.rowCount;
sheet.getRange(`B${lastRow + 1}`).formulas = [["=SUM(B2:B" + lastRow + ")"]];
await context.sync();
```

For filtering, utilize the `autoFilter` method with specific criteria.
```javascript
const sheet = workbook.worksheets.getActiveWorksheet();
const usedRange = sheet.getUsedRange();
usedRange.load("values");
await context.sync();
const filterCriteria = { criteria: "Engineering" };
usedRange.autoFilter.apply(filterCriteria);
await context.sync();
```

To create a named range based on specific conditions, use the `add` method with a conditional statement.
```javascript
const sheet = workbook.worksheets.getActiveWorksheet();
const usedRange = sheet.getUsedRange();
usedRange.load("values");
await context.sync();
const rows = usedRange.values;
const namedRange = [];
for (let i = 1; i < rows.length; i++) {
  if (rows[i][0] > 1000) {
    namedRange.push(rows[i]);
  }
}
const namedRangeAddress = "C2:E" + (namedRange.length + 1);
workbook.names.add("QuarterlySales", namedRangeAddress);
await context.sync();
```

For sorting, use the `sort` method with the correct column index and sorting order.
```javascript
const sheet = workbook.worksheets.getActiveWorksheet();
const usedRange = sheet.getUsedRange();
usedRange.load("values");
await context.sync();
const hdr = usedRange.values[0].map(h => String(h).toLowerCase().trim());
const col = hdr.indexOf("y");
if (col === -1) throw new Error("Column 'Y' not found.");
usedRange.sort.apply([{ key: col, ascending: true }]);
await context.sync();
```
