When updating comments, ensure the correct cell is targeted and use efficient methods.
```javascript
const sheet = workbook.worksheets.getActiveWorksheet();
const cell = sheet.getRange("A1");
cell.load("comment");
await context.sync();
if (cell.comment) {
  cell.comment.text = cell.comment.text + " - " + new Date().toLocaleDateString();
} else {
  cell.comment = { text: new Date().toLocaleDateString() };
}
await context.sync();
```
