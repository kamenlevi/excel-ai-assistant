When filtering data, use Office JS APIs such as `sheet.getRange().filter()` instead of throwing generic errors.
To add comments to the correct cell, ensure the row index is accurate by using `sheet.getRange(`A${maxRowIndex + 1}`)` without the extra `+ 1`.
When updating comments, target the correct cell range, such as `sheet.getRange("A1")`, and use the `comments` property to access and modify existing comments.
CODE_JS:
```javascript
const sheet = workbook.worksheets.getActiveWorksheet();
const filter = sheet.getRange("Department").filter.apply({
  criteria: "Engineering"
});
```
