To set page layout for multiple worksheets, iterate through worksheets and apply settings:
```javascript
const worksheets = workbook.worksheets;
worksheets.load("items");
await context.sync();
worksheets.items.forEach((sheet) => {
  if (sheet.getRange("A1").values[0][0] === "Draft") {
    sheet.pageSetup.orientation = Excel.PageOrientation.portrait;
  } else {
    sheet.pageSetup.orientation = Excel.PageOrientation.landscape;
  }
});
```
